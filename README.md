# TaskSpike

A kanban-style todo board where completed tasks go on a receipt spike.

The To Do and In Progress columns are styled like the order rails in a restaurant kitchen. When you finish a task, you drag it to the spike in the corner and it slides down onto the stack. Once the spike fills up (8 tickets), the whole stack gets cleared to the bin.

Built as a skeuomorphic design experiment. Single HTML file, no dependencies, localStorage only.

**[Live demo](https://shaynelarocque.github.io/taskspike/)**
